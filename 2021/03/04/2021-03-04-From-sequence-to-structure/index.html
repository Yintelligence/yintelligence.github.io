How do we define secondary structure classes?

  • Hydrogen bonding 1:10:48

    • Hydrogen on electronegative atom (N, O) has partial positive charge
    • Can form a hydrogen bond with another electronegative atom

  • Regular patterns of hydrogen bonding in secondary structure elements 1:16:35

  • DSSP (hydrogen bond estimation algorithm) 1:17:54

    • 1983, in Pascal

    • Simple definition for backbone hydrogen bond

      • r is distance
      • N, H backbone atoms from residue i
      • O, C backbone atoms from residue j

    • Assigns secondary structure based on patterns

      • α helix (H), i → i+4
      • β strand (E), long set of connected hydrogen bonds
      • 310 helix (G), i → i+3
      • And others…

  • Look at the local conformation of the amino acid.

Protein secondary structure prediction

Q3 measure 35:35

\[ Q_3 = \frac{N_{residues\_correctly\_predicted}}{N_{residues\_total}} \]

  • Random prediction should give 0.33
  • Real random prediction gives 0.38
    • Frequencies of secondary structure are not the same
  • Bad predictions can give high scores ...

Matthews correlation coefficient (MCC) 40:48

  • Predicted for each secondary structure state
  • Uses false/true positives and false/true negative

\[ MCC = \frac{TP×TN-TP×FN}{\sqrt{(TP+FP)(TP+FN)(TN+FP)(TN+FN)}} \]

  • Sensitivity (hit rate, recall)

    TPR = TP / (TP + FN)

  • Specificity

    SPC = TN / (FP + TN)

  • Report MCC always per secondary structure class
  • Multi-class definition also available

\[ MCC_{multiclass} = \frac{\sum_k\sum_l\sum_mC_{kk}C_{lm}-C_{kl}C_{mk}}{\sqrt{\sum_k(\sum_lC_{kl})(\sum_{k'|k'≠k}\sum_{l'}C_{k'l'})}\sqrt{\sum_k(\sum_lC_{lk})(\sum_{k'|k'≠k}\sum_{l'}C_{l'k'})}} \]

Chou-Fasman 54:42

  • Original attempt at predicting secondary structure
  • Chou and Fasman (1970s)
  • Based on very limited number of protein structures

Statistical propensity for each amino acid (R) to adopt a certain secondary structure (S)

Chou-Fasman algorithm
  • Assign α/β propensities to each amino acid

  • Find region with 4/6 connected residues pα > 1.0
    • Extend on both ends until:
      • 4 connected residues have average pα < 1.0 → End of α helix
    • Total segment longer than 5 residues, average pα > pβ ?
      • Assign as α helix

  • Find region with 3/5 connected residues pβ > 1.0
    • Extend on both ends until:
      • 4 connected residues have average pβ < 1.0 → End of β sheet
    • Average pβ > 1.05 and average pβ > pα?
      • Assign as β sheet
  • Repeat for whole sequence
  • If α/β overlap, choose which one by highest average p

Overall Chou-Fasman algorithm:

  • Based on statistics
  • Very simple
    • Per-residue, no residue interactions included
    • Algorithm cutoffs are pretty arbitrary
  • But gives better than random results!
    • Q3 score of 42-52% depending on protein family

GOR method 1:11:22

  • Evolved from GOR to GOR-V

    • Jean Garnier, David Osguthorpe, Barry Robson

  • Based on information theory approach (Bayes)

    • Self-information
    • Directional information
    • Pair information

  • Clearly defined model to get to results

  • Self-information (related to Chou-Fasman)

  • Self-information, more general

Calculated for the three conformations, the highest value of Eq. (2) for one of the conformations S will be the predicted conformation and will be the propensity for that residue to be in that conformation, usually expressed in centinat units when natural logarithms are used.

  • Include local sequence information
    • How often do you see a residue in relation to a secondary structure element at position j?

\[ I(ΔS_j;R_1,..., R_n ) = \log[P(S_j , R_1,..., R_n )/ P(n − S_j , R_1,..., R_n )] +log[P(n − S)/ P(S)] \]

Secondary structure for unknown sequence

Zpred

  • Based on GOR
    • Uses homologous sequence information
      • Averaging of prediction per residue
      • Calculates conservation value
  • Some amino acids are more similar to each other than others…
    • Related to physical characteristics (of the side chain), not DNA sequence mutation variation
  • Amino acid characteristics
  • Conservation value
    • Zvelebil conservation number (Cn)
      • 1.0 for identical residue
      • 0.9 for residues with same characteristics (e.g. I and L)
      • Subtract 0.1 for every different characteristic
        • R positive, polar
        • V aliphatic, hydrophobic, small
        • 5 differences; conservation 0.9 – (0.1 x 5) = 0.4
      • Gaps get all characteristics – Results in low conservation value!
  • The results…
  • Q3 ranges from 44.9% (allβ) to 76.9% (allα)
  • Results for all β are worse than GOR IV…

Hidden Markov model

  • Discrete Markov process
  • Transition probabilities expressed in transition matrix

Discrete Markov process

  • Probability for Sun-cloud-rain-rain-sun?

  • First have to define starting state (sun), then: \[ P = P(sun |sun)× P(cloud |sun)×P(rain | cloud)× P(rain |rain)× P(sun |rain) P = 0.50 × 0.30 ×0.35× 0.40 × 0.10 = 0.0021 \]

  • Can get probability for sequence of related events

    • Dependence only on previous state(s)!
    • Markov chain

Hidden Markov Model (HMM)

  • Define:

    • Hidden states (N=3) \[ S = \{S_1, S_2,..., S_N\} \]

    • State of model at step n \[ q_n \]

    • Observations (M=20) \[ V = \{v_1, v_2,..., v_m \} \]

    • State transition matrix \[ A = \{p_{i|j}\}\\ p_{i|j} = P(q_{n+1} = i | q_{n} = j),1≤ i, j ≤ N \]

    • Emission distribution \[ E = \{e_j(k)\}\\e_j(k) = P(V_kat n | q_n = S_{j});1≤ j ≤ N,1≤ k ≤ M \]

    • Initial state distribution \[ π= π_{i}\\ π_{i} = P(q_1 = S_j),1≤ j ≤ N \]

  • 3-state secondary structure HMM
    • Produce observation sequence
      • Choose an initial state Sj according to π; step q1
      • Choose an emission observation vk based on state
      • Proceed to step qn+1, pick state Si using transition probabilities from matrix A
      • Again emission, proceed or stop, …
  • Have to ‘recover’ the hidden state probabilities from a known observed sequence
    • Calculate all state sequences of sequence length T?
      • 2TNT calculations; not efficient
    • Forward algorithm
      • Start with first amino acid in sequence
        • Calculate probabilities for observing next amino acid in sequence for each secondary structure combination
      • N2T calculations
  • Hidden states model can be made more complex to take more variables into account
    • Hydrophobicity, charge bias, helix length, …
    • Incorporate homologous sequence
  • For example HHPred
